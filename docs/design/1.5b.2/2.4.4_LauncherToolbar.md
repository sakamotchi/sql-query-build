## 実装内容

### ファイル作成: `app/components/launcher/LauncherToolbar.vue`

```vue
<script setup lang="ts">
const emit = defineEmits<{
  newConnection: []
  refresh: []
  toggleView: []
}>()

const viewMode = ref<'grid' | 'list'>('grid')

const handleNewConnection = () => {
  emit('newConnection')
}

const handleRefresh = () => {
  emit('refresh')
}

const handleToggleView = () => {
  viewMode.value = viewMode.value === 'grid' ? 'list' : 'grid'
  emit('toggleView')
}
</script>

<template>
  <div class="flex flex-col sm:flex-row justify-between items-start sm:items-center gap-4">
    <!-- 左側: タイトルと統計 -->
    <div>
      <h2 class="text-3xl font-bold text-gray-900 dark:text-white">
        接続一覧
      </h2>
      <p class="text-sm text-gray-500 dark:text-gray-400 mt-1">
        <slot name="stats">
          <!-- デフォルトの統計情報 -->
        </slot>
      </p>
    </div>

    <!-- 右側: アクションボタン -->
    <div class="flex gap-2">
      <!-- 表示切り替え -->
      <UButton
        :icon="viewMode === 'grid' ? 'i-heroicons-list-bullet' : 'i-heroicons-squares-2x2'"
        variant="outline"
        color="gray"
        @click="handleToggleView"
      >
        {{ viewMode === 'grid' ? 'リスト' : 'グリッド' }}
      </UButton>

      <!-- リフレッシュ -->
      <UButton
        icon="i-heroicons-arrow-path"
        variant="outline"
        color="gray"
        @click="handleRefresh"
      >
        更新
      </UButton>

      <!-- 新規接続 -->
      <UButton
        icon="i-heroicons-plus"
        color="primary"
        size="lg"
        @click="handleNewConnection"
      >
        新規接続
      </UButton>
    </div>
  </div>
</template>
```

## Events

| Event | Payload | 説明 |
|-------|---------|------|
| `newConnection` | - | 新規接続ボタンクリック時 |
| `refresh` | - | 更新ボタンクリック時 |
| `toggleView` | - | 表示切り替えボタンクリック時 |

---

## Slots

| Slot | 説明 |
|------|------|
| `stats` | 統計情報表示エリア（例: "全15件 • 開発: 5件 • 本番: 3件"） |

---

## 機能

### 1. 新規接続ボタン
- プライマリカラー
- アイコン付き
- クリックで新規接続フォームへ遷移

### 2. 更新ボタン
- グレーアウトライン
- クリックで接続リストを再読み込み

### 3. 表示切り替えボタン
- グリッドビュー ⇔ リストビュー切り替え
- アイコンが現在と逆のモードを表示
