## 目的

環境バッジコンポーネントを実装（再利用可能なバッジUI）

---

## 実装内容

### ファイル作成: `app/components/common/EnvironmentBadge.vue`

```vue
<script setup lang="ts">
import type { Environment } from '~/types'

const props = defineProps<{
  environment: Environment
  size?: 'sm' | 'md' | 'lg'
  showIcon?: boolean
}>()

const { environmentLabel } = useEnvironment()

const environmentColors = {
  development: 'green',
  test: 'blue',
  staging: 'amber',
  production: 'red'
}

const environmentIcons = {
  development: 'i-heroicons-code-bracket',
  test: 'i-heroicons-beaker',
  staging: 'i-heroicons-cloud',
  production: 'i-heroicons-server'
}
</script>

<template>
  <UBadge
    :color="environmentColors[environment]"
    :size="size || 'md'"
    variant="solid"
  >
    <div class="flex items-center gap-1">
      <UIcon
        v-if="showIcon"
        :name="environmentIcons[environment]"
      />
      <span>{{ environmentLabel }}</span>
    </div>
  </UBadge>
</template>
```

## Props仕様

| Prop | 型 | 必須 | デフォルト | 説明 |
|------|---|------|-----------|------|
| `environment` | `Environment` | ✓ | - | 環境タイプ |
| `size` | `'sm' \| 'md' \| 'lg'` | - | `'md'` | バッジサイズ |
| `showIcon` | `boolean` | - | `false` | アイコン表示 |

---

## 環境別カラー・アイコン

| 環境 | カラー | アイコン |
|------|--------|---------|
| development | green | code-bracket |
| test | blue | beaker |
| staging | amber | cloud |
| production | red | server |

## 使用例
