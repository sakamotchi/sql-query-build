## 実装内容

### ファイル作成: `app/components/common/EnvironmentIndicator.vue`

```vue
<script setup lang="ts">
import type { Environment } from '~/types'

const props = defineProps<{
  environment: Environment
  position?: 'top' | 'left' | 'bottom' | 'right'
}>()

const { environmentColors } = useEnvironment()

const positionClasses = computed(() => {
  const positions = {
    top: 'top-0 left-0 right-0 h-1',
    left: 'top-0 left-0 bottom-0 w-1',
    bottom: 'bottom-0 left-0 right-0 h-1',
    right: 'top-0 right-0 bottom-0 w-1'
  }
  return positions[props.position || 'top']
})
</script>

<template>
  <div
    class="absolute"
    :class="[positionClasses, `env-indicator-${environment}`]"
    :style="{
      backgroundColor: environmentColors.primary
    }"
  />
</template>

<style scoped>
.env-indicator-development {
  box-shadow: 0 0 8px rgba(76, 175, 80, 0.5);
}

.env-indicator-test {
  box-shadow: 0 0 8px rgba(33, 150, 243, 0.5);
}

.env-indicator-staging {
  box-shadow: 0 0 8px rgba(255, 152, 0, 0.5);
}

.env-indicator-production {
  box-shadow: 0 0 8px rgba(244, 67, 54, 0.5);
}
</style>
```

## Props仕様

| Prop | 型 | 必須 | デフォルト | 説明 |
|------|---|------|-----------|------|
| `environment` | `Environment` | ✓ | - | 環境タイプ |
| `position` | `'top' \| 'left' \| 'bottom' \| 'right'` | - | `'top'` | インジケーター位置 |

---

## スタイル仕様

- トップ/ボトム: 高さ1px、横幅100%
- 左/右: 幅1px、縦幅100%
- 環境別にbox-shadowで発光効果

## 使用例

```vue
<!-- カードの上部に環境インジケーター -->

  <EnvironmentIndicator :environment="connection.environment" position="top" />
  <div class="p-4">
    <!-- カードコンテンツ -->
  </div>
</div>
```
