## 実装内容

### ファイル構成

**`app/pages/settings.vue`**

### ページ仕様

- **パス**: `/settings`（ランチャーからの遷移を想定）
- **役割**: 一般設定・セキュリティ設定・アプリ情報をタブで切り替え表示
- **初期状態**: `general`タブを選択し、設定/セキュリティストアを読み込む
- **レイアウト**: 環境ヘッダー + コンテンツ（最大幅5xl、余白: px-4/py-8）

### 実装コード

```vue
<script setup lang="ts">
import { storeToRefs } from 'pinia'

type SettingsTab = 'general' | 'security' | 'about'

const settingsStore = useSettingsStore()
const securityStore = useSecurityStore()
const { currentEnvironment } = useEnvironment()

const { loading: settingsLoading, error: settingsError } = storeToRefs(settingsStore)
const { loading: securityLoading, error: securityError } = storeToRefs(securityStore)

const tabs: { key: SettingsTab; label: string; icon: string }[] = [
  { key: 'general', label: '一般設定', icon: 'i-heroicons-cog-6-tooth' },
  { key: 'security', label: 'セキュリティ', icon: 'i-heroicons-lock-closed' },
  { key: 'about', label: 'について', icon: 'i-heroicons-information-circle' }
]

const selectedTab = ref<SettingsTab>('general')
const isLoading = computed(() => settingsLoading.value || securityLoading.value)
const errorMessage = computed(() => settingsError.value || securityError.value)

onMounted(async () => {
  await Promise.allSettled([
    settingsStore.loadSettings(),
    securityStore.loadSettings()
  ])
})
</script>

<template>
  <div class="min-h-screen bg-gray-50 dark:bg-gray-900">
    <EnvironmentHeader :environment="currentEnvironment" :show-toggle="true" />

    <main class="max-w-5xl mx-auto px-4 py-8 space-y-6">
      <div class="flex items-center justify-between gap-4">
        <div>
          <p class="text-sm text-gray-500 dark:text-gray-400">アプリ全体の動作を管理します</p>
          <h1 class="text-3xl font-bold text-gray-900 dark:text-white">設定</h1>
        </div>

        <UButton to="/" variant="ghost" color="gray" size="sm">
          ホームに戻る
        </UButton>
      </div>

      <UAlert
        v-if="errorMessage"
        color="red"
        variant="soft"
        icon="i-heroicons-exclamation-triangle"
        title="設定の読み込みに失敗しました"
      >
        {{ errorMessage }}
      </UAlert>

      <UTabs
        v-model="selectedTab"
        :items="tabs"
        class="bg-white dark:bg-gray-800 rounded-xl shadow-sm border border-gray-200 dark:border-gray-700"
      >
        <template #item="{ item }">
          <div class="py-6 px-2">
            <div v-if="isLoading" class="flex justify-center py-6">
              <UIcon name="i-heroicons-arrow-path" class="w-6 h-6 animate-spin text-gray-500" />
            </div>

            <GeneralSettings v-else-if="item.key === 'general'" />
            <SecuritySettings v-else-if="item.key === 'security'" />
            <AboutSection v-else-if="item.key === 'about'" />
          </div>
        </template>
      </UTabs>
    </main>
  </div>
</template>
```

## 技術仕様

### ルーティング

- **URL**: `/settings`
- **遷移導線**: ランチャーヘッダー／メニューからリンク、`to="/"`でホームへ戻る動線を提供

### UI構造

```
┌─────────────────────────────┐
│ EnvironmentHeader            │
├─────────────────────────────┤
│ ページヘッダー（タイトル/戻る）│
│ エラーアラート                │
│ ┌─────────────────────────┐ │
│ │ UTabs (general/security/ │ │
│ │ about) + タブ内容        │ │
│ └─────────────────────────┘ │
└─────────────────────────────┘
```

- 背景色: `bg-gray-50 dark:bg-gray-900`
- コンテンツ: `max-w-5xl mx-auto px-4 py-8`
- タブカード: `rounded-xl shadow-sm border`

### 状態・データ

| 変数 | 型 | 役割 |
|------|---|-----|
| `selectedTab` | `SettingsTab` | タブ切替（初期値: `general`） |
| `isLoading` | `boolean` | 設定/セキュリティ読み込みの複合ローディング |
| `errorMessage` | `string \| null` | いずれかのストアで発生したエラー |
| `tabs` | `SettingsTab[]` | ラベル・アイコン付きタブ定義 |

- 初回マウントで `settingsStore.loadSettings()` / `securityStore.loadSettings()` を非同期実行。
- ローディング中はスピナーを表示し、タブ内容は遅延表示（`v-if`）。
- エラー発生時は `UAlert` で理由を表示し、再読み込みはブラウザリロードで対応（Phase 4でリトライ導入予定）。

### 使用コンポーネント

- `EnvironmentHeader`（既存）: 環境表示・トグル
- `UTabs`（Nuxt UI）: タブ切替
- `UButton`（Nuxt UI）: ホーム戻り導線
- `UAlert`（Nuxt UI）: ロードエラー表示
- `UIcon`（Nuxt UI）: ローディングスピナー
- `GeneralSettings`（3.1.2）: 一般設定
- `SecuritySettings`（3.1.3）: セキュリティ設定
- `AboutSection`（静的情報、Phase 3.1で実装）

### 使用Composable / ストア

- `useSettingsStore()`  
  - `loadSettings()`：`get_settings` Tauriコマンドを呼び出し、`settings`を更新  
  - `loading` / `error` をローディング・エラーステートに利用
- `useSecurityStore()`  
  - `loadSettings()`：`get_security_settings`を呼び出し、セキュリティ設定を同期  
  - `loading` / `error` をローディング・エラーステートに利用
- `useEnvironment()`  
  - `currentEnvironment` を環境ヘッダーに供給

### データフロー

```mermaid
graph TD
    A[ページマウント] --> B[settingsStore.loadSettings]
    A --> C[securityStore.loadSettings]
    B --> D[settingsStore.loading/error]
    C --> E[securityStore.loading/error]
    D --> F[isLoading,errorMessage]
    E --> F
    F --> G[UTabsコンテンツ表示/スピナー]
    G --> H[GeneralSettings | SecuritySettings | AboutSection]
```
