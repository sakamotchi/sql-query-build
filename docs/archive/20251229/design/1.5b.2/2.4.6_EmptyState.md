## 目的

空状態表示コンポーネントを実装（接続が0件の場合の表示）

---

## 実装内容

### ファイル作成: `app/components/launcher/EmptyState.vue`

```vue
<script setup lang="ts">
const props = defineProps<{
  icon?: string
  title?: string
  description?: string
  actionLabel?: string
  actionIcon?: string
}>()

const emit = defineEmits<{
  action: []
}>()

const handleAction = () => {
  emit('action')
}
</script>

<template>
  <div class="flex flex-col items-center justify-center py-16 px-4">
    <!-- アイコン -->
    <UIcon
      :name="icon || 'i-heroicons-inbox'"
      class="text-6xl text-gray-400 mb-4"
    />

    <!-- タイトル -->
    <h3 class="text-xl font-semibold text-gray-700 dark:text-gray-300 mb-2">
      {{ title || '接続が見つかりませんでした' }}
    </h3>

    <!-- 説明 -->
    <p class="text-gray-500 dark:text-gray-400 text-center max-w-md mb-6">
      {{ description || '新しい接続を追加するか、検索条件を変更してください。' }}
    </p>

    <!-- アクションボタン -->
    <UButton
      v-if="actionLabel"
      :icon="actionIcon || 'i-heroicons-plus'"
      color="primary"
      size="lg"
      @click="handleAction"
    >
      {{ actionLabel }}
    </UButton>

    <!-- デフォルトスロット（カスタムコンテンツ用） -->
    <div v-if="$slots.default" class="mt-6">
      <slot />
    </div>
  </div>
</template>
```

## 使用例

### 基本的な使い方

```vue
<EmptyState />
```

### カスタマイズ

```vue
<EmptyState
  icon="i-heroicons-magnifying-glass"
  title="検索結果が見つかりません"
  description="別のキーワードで検索してみてください。"
  action-label="新規接続を追加"
  @action="navigateToForm"
/>
```

### スロットでカスタマイズ

```vue
<EmptyState
  title="接続がありません"
  description="まずは最初の接続を作成しましょう。"
>
  <template #default>
    <div class="flex gap-2">
      <UButton>クイックスタート</UButton>
      <UButton variant="outline">ドキュメントを見る</UButton>
    </div>
  </template>
</EmptyState>
```
